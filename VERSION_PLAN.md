# 《古墓不让进》Web原型 版本计划

依据策划案《古墓不让进2.0》（轻射击PVE搜打撤RPG）拆分三个版本：

## V1 核心单局循环 ✅目标
- 第一人称越肩跟随相机（策划案"镜头"节）
- 房间制关卡：房间链动态生成（战斗/宝藏/祭坛/通道/撤离点），自动前进
- 四状态机：向前移动(自动) / 战斗(手动瞄准射击) / 选择(点击岔路门) / 搜索(自动读条)
- 射击：鼠标瞄准+点击开枪、弹夹与换弹(自动+R)、分部位伤害(爆头×3)
- 敌人：普通丧尸（近身扣血）
- 神智：进新房间扣神智，归零强制撤离
- 撤离点：撤离结算(战利品入账) 或 继续深入(敌人增强)
- 死亡惩罚：丢失本局全部战利品（搜打撤核心反馈）
- 微恐氛围：黑暗+手电筒+火把光源
- HUD：血量/神智/弹夹/金币/深度/准星/状态提示

## V2 搜打撤决策强化
- 背包格位管理（容量取舍）+ 消耗品独立槽（医疗/手雷手动使用）
- 敌人类型扩展：高速/远程/重甲/精神怪（扣神智）/精英
- 祭坛局内增益、强化弹药
- 安全屋检查点（再入从安全屋出发）
- 战前情报提示（精英/BOSS房进入前预警）
- 收藏品（品质区分）

## V3 章节/养成/助手
- 章节制+多主题（金字塔/玛雅神庙/秦陵）+ 简易剧情
- BOSS房（分阶段机制）
- 局外养成：角色等级/武器改装/双武器携带
- 助手系统（宠物，主动+被动技能）
- 收藏品图鉴
- 异步排行（可选）

## V5 肉鸽化深入抉择 ✅本版
- 深入契约：撤离点"继续深入"时 2 选 1（可婉拒）的增益+代价契约，跨次叠加
  - 血祭（伤害↑/最大生命↓）、贪婪（撤离价值↑/神智上限↓）、亡命（移动搜索↑+手雷/受伤↑）
- 玩家新增受伤倍率(dmgTakenMult)、撤离结算价值倍率(lootValueMult)
- 结算界面展示契约价值换算明细

## V6 第三人称射击 ✅本版
- 第三人称全身角色（世界坐标模型）+ 后上方相机；角色占画面下部小幅
- 战斗自动瞄准最近敌人身体并自动开火（不暴击）；player.setAimTarget 控制朝向
- 手动 ADS：按住进入，FOV 68→44、中心准星、摄像机中心射线落点（可暴击）
- player.aiming / muzzlePos / faceYaw 等新状态；hud.setAds 中心准星

每版本完成后备份至 archive/v{n}/。
