# 《古墓不让进》需求日报（2026-06-22）

> 本日围绕原型的射击手感、相机表现、触屏 HUD、背包系统、房间与经济系统等进行了多轮迭代，以下按系统归类汇总当日全部需求。

## 【瞄准与射击】
1. 长按后才进入手动瞄准，短按用于点击/拾取（阈值由 0.5s 调到 0.15s），解决想射击时的卡顿。
2. 自动瞄准时角色要正确面向开枪目标。
3. 手动瞄准时只进入瞄准状态，准星指向敌人时才开火，不对空乱放。
4. 多次扩大手动瞄准左右偏转角（0.55→1.0→1.5），并增大向上抬枪角度。
5. 索敌机制：优先锁定最近怪物，锁定后除非该怪死亡不自动切换；点击其他丧尸可手动切换目标。
6. 射击命中造成减速（0.1× 速度，时长 1s→0.5s）；取消击退效果；精英怪免疫减速。

## 【相机与角色表现】
1. 未操控（闲置）时角色固定在画面左下角，使用固定俯瞰镜头。
2. 操控时相机随角色朝向旋转（等效为角色的子物体）。
3. 角色上半身可弯腰俯仰、整体可旋转来调整瞄准。
4. 减少瞄准时 FOV 收缩幅度（44→58），保留更多视野。
5. 玩家停下位置更靠近进入的房门，保证镜头朝前不转向（交互物都在正前方）。

## 【底部 HUD / 触屏操作】
1. 整体改为手机触屏点击操作。
2. 左下角背包按钮召出背包；中间武器1/武器2按钮（点击切换、点当前武器换弹、选中外框标识）、手雷按钮（投向怪物最密集处）、治疗按钮。
3. 武器按钮按实际携带数量动态生成。
4. 两个药品栏合并为一个物品栏 + 下拉展开显示所有消耗品；主按钮按生命/理智百分比显示并优先使用"最需要"的消耗品。
5. 新增暂停按钮，可暂停游戏进度。

## 【背包系统】
1. 武器、子弹、手雷、药品统一为背包物品，HUD 仅显示其余量。
2. 新增 2 个武器栏；把武器拖入武器栏即装备并同步底部武器 HUD；未装备时默认手枪。
3. 塔克夫/三角洲式网格背包：一个物品占多个格子；点击物品交互（武器装备、消耗品使用、所有物品可丢弃）。
4. 物品可拖动整理；拖到已被占用的格子上时与该物品互换位置。
5. 打开背包时游戏时间暂停。
6. 背包界面显示当前局内金币；右上角显示"局内金币 + 背包物品总价值"的合计。

## 【武器与弹药】
1. 新增武器 AK 作为武器2。
2. 子弹改为消耗背包库存，不再无限。
3. 区分步枪弹与手枪弹（分别消耗）；后续改为：手枪无限备弹、删除"手枪子弹"物品，仅保留步枪弹/霰弹。

## 【房间系统】
1. 增大单个房间的纵深（前后空间更大），左右宽度不变。
2. 每完成一个房间都要进入选房阶段，不选则不会自动前进；随机 2 选 1 或 3 选 1。
3. 修复 3 选 1 选择 UI 在 9:16 竖屏下溢出的问题。
4. 房型精简为 7 类：BOSS（撤离点，房前必有商店）、补给、宝石、神之祝福、神之诅咒、安全屋、商店。
5. 房间改名：宝石/钻石之屋→宝藏之屋；神之祝福→神明祭坛→祝福祭坛；神之诅咒→恶魔祭坛→诅咒祭坛。
6. 补给/宝石（战斗房）需清光所有丧尸后才能搜索；可交互物品高亮、刷新在镜头视野内、带品质区分。
7. 安全屋：无需先点篝火即可搜补给；需搜完物资且点击篝火后才自动前往下一房间。
8. 祭坛/篝火等资源改为"门口停下→点击交互→再继续移动"。

## 【消耗品】
1. 新增 6 种消耗品：
   - 荷鲁斯之泪：理智 +45，生命 -30
   - 过期镇定剂：理智 +15，生命 -5
   - 奥西里斯之脂：生命 +35，理智 -20
   - 木乃伊绷带：生命 +30，理智 -10
   - 野蜂蜜：理智 +15，生命 +10
   - 肾上腺素：血量/理智临时 +30，之后 3 个房间每次各 -10

## 【祝福 / 诅咒 / 增益】
1. 神之祝福改为三选一供玩家选择；为纯增益、无代价，且界面不显示"无代价"字样。
2. 神之诅咒祭坛为多选抉择（以理智/血量或其上限为代价换增益，也可不交换）。
3. 祝福去除「疾风之足」；新增【头脑冷静】（进房消耗理智永久 -1，最多 3 层）、【死神之力】（击杀后 15% 概率回 5 生命）、【神枪手】（爆头伤害 +10%）。
4. 下调祝福数值。
5. 三选一界面标题改为对应文案（不再错误显示"选择路线"）。
6. 核查祝福/诅咒为局内临时增益：每次失败/撤离/退出清除、每局重新获取（确认现逻辑已满足，无需改动）。

## 【经济：金币 / 商店】
1. 击杀丧尸随机掉落金币（金色光点、数量随机），点击拾取时金币图标像物品一样飞入背包按钮。
2. 商店只用局内背包金币购买；撤离时局内金币结算入金库。
3. 商店刷新规则：不刷在前 5 个房间、两个商店间隔 ≥5 个房间、BOSS 房前必刷一次；并提高随机出现概率。
4. 新增花金币刷新商店的机制。

## 【敌人】
1. 怪物血量整体 ×2、移动速度 ×1.5。
2. 怪物生成位置更靠近房间后方。
3. 仅保留减速效果、取消击退；精英怪免疫减速。

## 【理智系统】
1. 理智消耗机制：由"进房扣除"→"随战斗时间持续扣除"→最终改为"进入会刷怪的房间时一次性扣除"，并将扣除值标注在选房卡片上。
2. 加大理智消耗幅度。
3. 神智崩溃前用屏幕暗角明显预警（区别于低血红色，改为蓝色）。

## 【搜刮 / 掉落表现】
1. 搜索容器时：容器顶部显示圆形进度 UI + 绕圆心旋转的搜索图标，进度条颜色为容器内物品的颜色，转满获得物品。
2. 搜刮完成时物品图标：增大、去掉外框圆环、改用圆形且四周渐隐的品质色色块，并沿曲线飞入左下角背包按钮后消失。

## 【物品 / 品质】
1. 物品引入品质区分（普通 / 稀有 / 史诗 / 传说）。
2. 删除易与金币混淆的物品"古旧铜币"，替换为"青铜短剑"。
