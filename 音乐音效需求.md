# 音乐音效需求清单

## 当前状态

- 现在代码里使用 `src/audio.ts` 的 WebAudio 程序化占位音效，已经覆盖枪声、换弹、命中、怪物低吼、Boss 吼叫、门声、搜索、拾取、祭坛、神智受击、撤离、死亡和低神智心跳。
- `startAmbient()` 当前没有播放背景音乐，只保留事件音效与低神智心跳。
- `Assistant` 主动技能会调用 `audio.thunderLike()`，但 `AudioFX` 里还没有这个方法；后续需要补一个程序化电击音或接素材。

## 我可以继续完成的部分

- 建一个统一 AudioManager：音乐、环境、战斗、UI、SFX 分组音量。
- 接入外部素材加载：支持 `ogg/mp3/wav`，带预加载和失败回退。
- 为音乐做淡入淡出和场景切换：地图、探索、战斗、Boss、商店、安全屋、结算。
- 为短音效做随机音高、随机音量、冷却，避免同一音效连发刺耳。
- 补齐当前缺口：`thunderLike()`、金币拾取独立音效、地图/背包/商店 UI 音效触发。
- 在素材没到位前继续用程序化占位音，保证每个触发点都能先跑通。

## 素材建议

- 短音效：优先 `wav` 或 `ogg`，单个 0.1-3 秒。
- BGM/环境循环：优先 `ogg`，也可 `mp3`，需要无缝循环版本。
- 统一响度，避免爆音；素材尾部不要有明显空白。
- 命名建议按模块：`ui_click_01.ogg`、`weapon_pistol_fire_01.ogg`、`amb_tomb_loop_01.ogg`、`boss_sandlord_roar_01.ogg`。
